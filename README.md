## Monologue是什么？

Monologue 是一个实验性的多Agent协作工作平台。主要功能包括：

- **任务编排**：将复杂需求拆解为可并行的子任务，并调度合适的 Agent。
- **自进化智能体**：让每个 Agent 拥有独立记忆和进化能力，执行后自动复盘优化。
- **动态 Agent 生成**：通过自然语言即可快速定制新 Agent，即插即用。
